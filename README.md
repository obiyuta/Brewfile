# Brewfile

## Requirements

- Homebrew
- [Homebrew Bundle](https://github.com/Homebrew/homebrew-bundle)

## Usage

```
brew bundle install
```

## Fonts

### [ricty](https://github.com/sanemat/homebrew-font)

```
cp /usr/local/Cellar/ricty/{version}/share/fonts/Ricty*.ttf ~/Library/Fonts/
fc-cache -vf
```
